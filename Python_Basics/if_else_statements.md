# IF-ELSE Statements in Python

## 1. Simple IF Statement

Syntax:

```python
if condition:
    statement
## Example
age = 20

if age >= 18:
    print("You are eligible to vote")
    
## 2. IF ELSE Statement
if condition:
    statement
else:
    statement
## Example
num = 10

if num % 2 == 0:
    print("Even number")
else:
    print("Odd number")

## 3. IF ELIF ELSE Statement

marks = 75

if marks >= 90:
    print("Grade A")
elif marks >= 70:
    print("Grade B")
elif marks >= 50:
    print("Grade C")
else:
    print("Fail")
